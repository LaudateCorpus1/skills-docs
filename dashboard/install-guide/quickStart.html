# Quick Start

Overall steps to getting started with SkillTree are:

  1. Prerequisites - don't worry there isn't much here
  2. Install Dashboard and create your project
  3. Integrate User Skills Display into your application.
  4. Report Skill Events using one of our drivers.

# Prerequisites

Dashboard requires Java Runtime Environment 11 +, we suggest using OpenJDK packages. Please note that this is not relevant if you elect to go with the Docker based install.

# Install Dashboard

You can utilize a pre-built docker image to run a production grade Skills Dashboard or you can download the latest distribution.
